# 3D 模型查看器

基于 Three.js 的 3D 模型查看器，支持 GLB/GLTF 模型加载和动画播放。

## 功能特性

- 📁 上传本地 GLB/GLTF 模型文件
- 🔗 从 URL 加载模型
- 🎬 自动播放模型动画
- 🎮 动画选择器切换不同动画
- ⏯️ 播放/暂停控制
- 🔄 自动旋转开关
- 📐 线框模式切换
- 🖱️ 拖拽旋转、滚轮缩放、右键平移

## 使用方法

### 本地运行

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080

### 上传模型

1. 点击"上传模型"按钮
2. 选择本地 GLB/GLTF 文件
3. 模型自动加载并播放动画

### 从 URL 加载

1. 点击"上传模型"按钮
2. 在输入框中输入模型 URL
3. 点击"从 URL 加载"

## 技术栈

- Three.js 0.160.0
- HTML5 Canvas
- WebGL

## 支持的模型格式

- GLB (.glb)
- GLTF (.gltf)
